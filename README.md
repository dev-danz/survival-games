  <a href="https://skillicons.dev">
    <img src="https://skillicons.dev/icons?i=js,html,css," />

</p> Com o objetivo de praticar lógica, animação, spawn de inimigos, câmera, colisão e sistemas de upgrade.

O jogo é inspirado em jogos survival estilo VAMPIRE SURVIVORS onde o jogador precisa sobreviver o máximo de tempo possível enquanto enfrenta inimigos cada vez mais fortes.

---

## 🚀 Funcionalidades

- ✔ Movimentação com WASD
- ✔ Sistema de câmera seguindo o player
- ✔ Inimigos com diferentes tipos (normal, fast, tank, boss)
- ✔ Sistema de level e upgrades
- ✔ Tiros automáticos
- ✔ Circle shot (power up temporário)
- ✔ Power-up de maçã
- ✔ Árvores geradas aleatoriamente
- ✔ Colisão com cenário
- ✔ Barra de vida
- ✔ Sistema de pause / restart
- ✔ Menu inicial
- ✔ Partículas de explosão
- ✔ Spawn progressivo de inimigos

## 🌳 Mundo procedural

O mapa é gerado automaticamente com:

- grama em tiles
- árvores aleatórias
- spawn de inimigos fora da tela
- câmera dinâmica

---

## 🎮 Controles

| Tecla | Ação |
|--------|---------|
| W A S D | mover |
| SPACE | iniciar |
| P | pause |
| R | restart |
| 1 2 3 | escolher upgrade |
